MOODK Clothing Store 👕

MOODK is a modern clothing store website built as a team project. The website is designed to provide a smooth and simple shopping experience for users browsing different clothing categories.

📌 Project Overview

This project is a multi-page e-commerce frontend website that includes:

Home Page
Men Section
Women Section
Kids Section
Sale Section

Each page is designed to showcase products in a clean and user-friendly layout.

JavaScript (Vanilla JS)
